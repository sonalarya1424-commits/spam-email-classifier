# Spam Email Classifier

This is a beginner-friendly Machine Learning project that classifies messages as Spam or Not Spam.

## Project Description
The model is trained on an SMS dataset using the Naive Bayes algorithm. Text data is converted into numerical form using CountVectorizer.

## Features
- Text preprocessing
- Machine learning model training
- Accuracy evaluation
- Custom message prediction

## Accuracy
~98%

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
python3 spam_classifier.py
